What is BRICKS?</h2>

BRICKS is a list for all the domains blocked by the Chinese GFW.

Project URL: https://github.com/Leask/BRICKS</a>

What for?</h2>

  • In order to collect all the domains blocked by the GFW with the contributers all over the internet.</li>
  • To study how tight the restrictions are, in the Chinese internet.</li>
  • To log the data about this terrifying digital Berlin Wall as much as possible.</li>
  • Help you to walkaround the GFW with a proxy server.</li>
    </ul>

    Different from GfwList</h2>

    GfwList</a> is cool and effective, but is complex and inefficient currently. Now, I want to make it simple and work more smoothly.

    Installation</h2>

    $ git clone https://github.com/Leask/BRICKS.git</code></p>

    Uasge</h2>

    • List all websites blocked by Chinese government
      $ ./bricks list</code>
      or just simply $ cat gfw.bricks</code></li>

    • Add domain into BRICKS
      $ ./bricks add 'google.com'</code></li>

    • Remove domain from BRICKS
      $ ./bricks remove 'facebook.com'</code></li>

    • Make PAC file with BRICKS
      $ ./bricks makpac 'SOCKS5 127.0.0.1:8964; SOCKS 127.0.0.1:8964; DIRECT'</code></li>

    • Make PAC file and launch HTTP server
      $ ./bricks pacsrv 'SOCKS5 127.0.0.1:8964; SOCKS 127.0.0.1:8964; DIRECT'</code></li>

    • Show help infos
      $ ./bricks help</code></li>
      </ul>
