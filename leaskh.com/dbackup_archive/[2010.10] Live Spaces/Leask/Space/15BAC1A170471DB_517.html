Great UI Designer: Siker

6/4/2005 3:19:55 PM

Siker (思客) 是一名很出色的 UI (User Interface,用户界面) 设计师，是我崇拜的设计师之一，喜欢他那些精致的塑胶风格。

通过阅读 Siker 的 Blog 让我欣赏到很多他的精彩作品，还有看到了他出神入化的设计技巧和出众的才华。

Siker 的 Blog： http://www.siker.cn

近期中国的 UI 设计是一个比较热门的论题，看到现在国外软件的侵蚀，真的很希望更多像Siker 那样出色的国内 UI 设计师可以更加努力，在世界舞台展现自己的才华，让世界听到更多来自中国 UI 设计师的声音。

相比之下，自己的路还很漫长，真的很漫长。

A：其中有为了纪念 Photoshop CS2 发布而用 Photoshop CS2 手绘的 Kodak Z740 相机，已经达到可以乱真的精细度。

B：练习作品，为 QQ Music 设计的播放器 UI，同样很精致。
